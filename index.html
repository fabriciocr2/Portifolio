Meu Curriculum

Sobre Mim

Iniciei minha jornada no desenvolvimento web com uma fascinação pela forma como a tecnologia pode transformar ideias em interfaces visuais interativas. Ao longo do meu caminho, mergulhei nos fundamentos sólidos de HTML, CSS e JavaScript.

Em meus primeiros projetos, concentrei-me em construir páginas web simples, garantindo uma base sólida nas linguagens básicas. Explorei a estilização com CSS para criar layouts visualmente atraentes e comecei a dar vida às minhas páginas com JavaScript.

Acredito firmemente no poder da aprendizagem contínua. Cada desafio que encontro é uma oportunidade de crescimento, e estou sempre buscando maneiras de melhorar minhas habilidades e conhecimentos.

Estou animado com a perspectiva de crescer como desenvolvedor front-end e contribuir para a evolução da web. Se você está em busca de alguém dedicado, apaixonado e disposto a enfrentar desafios, estou pronto para a oportunidade. Obrigado por explorar um pouco mais sobre mim e meu percurso no mundo do desenvolvimento front-end!

Minhas Especialidades

HTML

Conhecimento amplo em estruturação de paginas com html,fazendo uso de tags e construçoes semanticas.

CSS

Em conjunto com o html adquiri uma boa pratica com css,sendo capaz de aplicar os mais belos estilos a paginas Web.

JavaScript

Bem como as outras duas skills o javascript se faz importante quando o assunto é adicionar dinamismo as paginas.
